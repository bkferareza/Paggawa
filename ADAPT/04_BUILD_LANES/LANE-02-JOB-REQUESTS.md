# LANE-02 - Job Requests

STATUS: VALIDATED

## Purpose

Allow residents in Paggawa Mobile and barangay staff in Paggawa Quest to create job requests.

## Required Inputs

- App Shell
- Domain Model
- Product Brief

## Intended Output

A resident can create a job request in Paggawa Mobile.

A barangay staff user can create a job request on behalf of a resident in Paggawa Quest.

Created jobs appear in relevant shared views.

## In Scope

- Paggawa Mobile create job form
- Paggawa Quest assisted create job form
- job request list
- resident-created request
- barangay-assisted request
- mock/local persistence

## Out of Scope

- payments
- real backend
- photo upload
- advanced validation
- exact address exposure

## Validation Criteria

- Resident can create a job request in Paggawa Mobile.
- Barangay staff can create a job request in Paggawa Quest.
- Created job appears in resident view.
- Created job appears in Paggawa Mobile worker nearby jobs view.
- Created job appears in Paggawa Quest quest board.
